# DEPStandardReader
A script to read a csv of multiple DEP Remediaiton Standards and report if an input is in excess of remediation standards

CSV should be in the format CAS No | Standard 1 | Standard 2 | ... etc
If you have chemicals that are not currently in the local CASN library (located in the CASN.py file) you can add it to your library if you dont wish to always have to type the CASN into the program.
